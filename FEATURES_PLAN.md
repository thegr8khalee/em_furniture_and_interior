# Implemented Features Only

This document lists only the features that are implemented.

## Phase 1: Public Website (Customer-Facing)
- Blog / Inspiration: blog model, list + detail pages, admin CRUD.
- FAQs: static or CMS-backed FAQ page.
- Book Consultation Page (Calendly): embed Calendly and capture lead metadata.

## Phase 2: Advanced Product Catalog
- Estimated delivery date on product page (lead time + shipping profile).
- Product comparison (compare list + compare view + API support).
- Recently viewed (client cache + optional user storage).

## Phase 3: E-Commerce Features
- Apply coupon (coupon model + validation + cart integration).
- Tax calculation (TaxJar/Avalara or rules engine).
- Guest checkout (order flow without account).
- Order summary preview (final pricing breakdown).
- Invoice download (PDF invoice generation).

## Phase 4: Customer Account Features
- Order history (list + detail pages).
- Track order (self-serve tracking).
- Reviews and ratings (verified-purchase gating + moderation).
- Notification center (in-app notifications).
- Loyalty points (earn/redeem rules + balance).

## Phase 5: Interior Design Module (Very Important)
### Consultation System
- Book online consultation.
- Video meeting integration (Calendly + Zoom/Meet).
- Choose designer.
- Upload room photos.
- Upload floor plan.
- Choose budget range.
- Style preference selector.

## Phase 6: Admin Panel (Super Admin)
### Order Management
- View all orders.
- Order status updates.
- Generate invoice.
- Invoice, receipt, quotation builder.

### Interior Management
- View room submissions.
- Designer performance analytics.

### Finance Module
- Revenue breakdown.
- Export accounting reports.

### Marketing Tools
- Coupon creation.
- Promo banners.
- Flash sales.

### Inventory and Warehouse
- Inventory tracking.

### SEO Fields per Product
- Add SEO metadata per product + schema JSON-LD.

## Phase 7: Analytics and Reporting
- Sales by category.
- Sales by region.
- Product performance.
- Designer performance.
- Customer lifetime value.
- Conversion funnel.

## Phase 8: Security and System Features
- Role-based access control.
- Audit logs.
- Activity tracking.
- Rate limiting.

## Suggested Service Integrations (Optional)
- Payments and invoices: payment gateways (Paystack, Flutterwave, manual bank transfer proof upload, Pay-by-transfer option at checkout, Stripe for international payments) + PDF generation.
- Taxes: TaxJar/Avalara.
